# Custom_Macropad
I made a Custom 16 key macropad with a trackball. It uses a Seeed studio Xiao Esp32-c3, a PMW3360DM-T2QU optical trackball sensor and some Linear mechanical switches. 

##To Use: 
1. Turn on Toggle the slide switch to turn on.
2. Connect: Use the USB-C port for a wired connection or pair via Bluetooth for wireless use.
3. Programming: program the 4x4 mechanical key matrix for custom shortcuts, or use the Code given in Code.cpp
4. Navigation: Use the trackball for cursor movement.
5. Turn off the macrppad to conserve power and remember to charge through the esp32-c3's usb-c port
